### Markdown syntax

## 1. 見出し
- # 見出し1（h1）
- ## 見出し2（h2）
- ### 見出し3（h3）
- #### 見出し4（h4）
- ##### 見出し5（h5）
- ###### 見出し6（h6）

## 2. 強調
- **太字** または __太字__
- *斜体* または _斜体_ (*ABC* or _ABC_)
- ~~打ち消し線~~

## 3. 番号なしリスト
- リスト1
  - ネスト1
  - ネスト2
- リスト2
  
  または
  
* リスト1
  * ネスト1
  * ネスト2
* リスト2

## 4. 番号付きリスト
1. 項目1
2. 項目2
3. 項目3
